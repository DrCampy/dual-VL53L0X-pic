2 sensors:
At start:
    Shutdown 1 sensor
    change it's address using general call address
    wake up both sensors
    Calibrate sensor 1
    Calibrate sensor 2


Config register :

Enable Left
Enable Right
Enable/disable XTalk compensation
Enable/Disable internal auto increments
Enable/Disable continuous measurement mode
Start conversion
Conversion finished flag

Interrupt mode (2 bits)
Duration of measurement (6 bits)

Change I2C address

autoIncrement / cycle :

Disabled
    Config:
        Write to/returns one config register at a time
        Writing:
            ADDR+R_NOT_W - Reg - Data[reg] - Reg - Data[reg] - P
        Reading :
            ADDR+R_NOT_W - Reg - RS - ADDR+R_NOT_W - Data[reg] - RS - ADDR+R_NOT_W - Reg - ADDR+R_NOT_W - Data[reg] - P //If trying to read more than one register the sensor will repeat the data in that register.

    Distance:
        Returns one distance at a time
        Reading:
            ADDR+R_NOT_W - Reg - RS - ADDR+R_NOT_W - Data - RS - ADDR+R_NOT_W - Reg - RS - ADDR+R_NOT_W - Data - P
        Writing:
            Not allowed.


Enabled
    Config:
        Write to/returns config registers by cycling through them.
        Writing:
            ADDR+R_NOT_W - Reg - Data[reg] - Data[reg+1] - Data[reg+2] - Data[reg] - ... - P //Cycles through the config registers, ommiting distance registers
        Reading :
            ADDR+R_NOT_W - Reg - RS - ADDR+R_NOT_W - Data[reg] - Data[reg+1] - Data[reg+2] - ... - P //Cycles through the config registers, ommiting distance registers

    Distance:
        Returns distances one after the other.
        Reading :
            ADDR+W - Reg - RS - ADDR+R - Data[reg] - Data[reg+1] - Data[reg+2] - Data[reg+3] - P //Cycle through the distance registers, ommiting config registers
        Writing:
            Not allowed.
