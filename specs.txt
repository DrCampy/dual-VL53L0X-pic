2 sensors:
At start:
    Shutdown 1 sensor
    change it's address using general call address
    wake up both sensors
    Calibrate sensor 1
    Calibrate sensor 2


Ext I2C specs:

Config register :
Needed functions:
Enable/disable
Enable Left
Enable Right
Enable interrupt
Choose Sampling Frequency
Change I2C address
Enable/disable XTalk compensation

autoIncrement / cycle :

Disabled
    Config:
        Write to/returns one config register at a time
        Writing:
            ADDR+R_NOT_W - Reg - Data[reg] - Reg - Data[reg] - P
        Reading :
            ADDR+R_NOT_W - Reg - RS - ADDR+R_NOT_W - Data[reg] - RS - ADDR+R_NOT_W - Reg - ADDR+R_NOT_W - Data[reg] - P //If trying to read more than one register the sensor will repeat the data in that register.

    Distance:
        Returns one distance at a time
        Reading:
            ADDR+R_NOT_W - Reg - RS - ADDR+R_NOT_W - Data[reg](L) - Data[reg+1](H) - RS - ADDR+R_NOT_W - Reg - RS - ADDR+R_NOT_W - Data[reg](H) - P
            //Sensor will respond with 2 bytes if reading L part register (L followed by H), 1 byte if reading H register)
            //If trying to read more than the expected number of bytes the sensor will cycle through the H and L registers of each distance.
        Writing:
            Not allowed.


Enabled
    Config:
        Write to/returns config registers by cycling through them.
        Writing:
            ADDR+R_NOT_W - Reg - Data[reg] - Data[reg+1] - Data[reg+2] - Data[reg] - ... - P //Cycles through the config registers, ommiting distance registers
        Reading :
            ADDR+R_NOT_W - Reg - RS - ADDR+R_NOT_W - Data[reg] - Data[reg+1] - Data[reg+2] - ... - P //Cycles through the config registers, ommiting distance registers

    Distance:
        Returns distances one after the other.
        Reading :
            ADDR+W - Reg - RS - ADDR+R - Data[reg](H) - Data[reg+1](L) - Data[reg+2]_H - Data[reg+3]_L - P //Cycle through the distance registers, ommiting config registers
        Writing:
            Not allowed.
